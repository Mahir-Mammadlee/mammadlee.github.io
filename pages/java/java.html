1. Java nədir

Xoş gördük.

Sizi bu səhifədə görməyimə məmnun oldum. Demək ki, öyrənməyə həvəslisiniz. İlkin məlumatlar olaraq paylaşımlarımdan yararlana və proqramlaşdırma sahəsinə ilkin addımlarınızı ata bilərsiniz. Bu sahədə heç nə bilməyən biri də paylaşımlarımdan bəhrələnə biləcək. Davamlı olaraq təməl səviyyədən yuxarı səviyyəyə qədər bilgilər paylaşacam. İndi isə keçək mövzumuza.

Proqramlaşdırmada ən çox istifadə olunan dillərdən biri 23 may 1995-ci ildə Sun Microsystems tərəfindən irəliyə sürülmüş, hazırda Oracle firmasının dəstək verdiyi açıq qaynaqlı Java dilidir. Əvvəlcə Oak ("palıd") adlandırılan bu dil James Gosling tərəfindən məişət cihazlarının proqramlaşdırılması üçün yaradılıb. Sonra isə bu ad Java ilə əvəz olundu və server proqram təminatının yazılması məqsədi ilə istifadə olunmağa başladı. Proqramlaşdırmaya yeni başlamaq üçün, işin məntiqini qavraya bilmək üçün, fikrimcə, ən ideal dildir. Bütün struktur bir neçə anlayışdan təşkil edilib və hər şey göz qabağında olduğu üçün anlaşılması asandır. Əməliyyat sistemlərindən asılı olmayaraq çalışa bilən struktura malikdir. Java’nın köhnə versiyası ilə yazılmış kod yeni versiyalarda da keçərlidir. Serverdə Java proqramının işini təmin etmək məqsədilə əlavə modullarının yüklənmə və qurulmasına ehtiyac olmur. Java dilinin kompilyasiyasını dəstəkləməyən köhnə versiyalı web browser ilə işləyən istifadəçilər bu dildə yaradılmış obyektləri görə bilmirlər.

JavaScript və Java adları biri-birinə bənzəsə də hər ikisi fərqli proqramlama dilidir və məqsədləri fərqlidir. Web tərəfdən yanaşsaq biri forntend developer’in, digəri isə backend developer’in istifadə etdiyi dildir. JavaScript ilə web saytımızın istifadəçiyə görünən HTML hisəsini emal edərək dizayn veririksə, Java ilə istifadəçinin daxil etdiyi məlumatları, sistemin gözlə görünməyən data hissəsini, məlumatların yönləndirilməsini emal edirik. Təbii ki, bu izah detallı və dəqiq deyil. Ümumi anlayış baxımından ilkin mərhələdə məsələyə izah etdiyim kimi yanaşmaq doğru olar deyə düşünürəm.

Java nə üçün gərəklidir, harada istifadə edə bilərəm deyirsinizsə: Java desktop, web, mobile, hybrid proqramlamada əhəmiyyətli rola malikdir. Xüsusilə mobile və web sahəsində, startup’larda geniş tətbiq olunur. Fotoda gördüyümüz cədvəl TIOBE Software BV company’yə əsasən 2020 və 2021-ci il üçün proqramlama dillərinin istifadə arealı cədvəlidir. Gördüyümüz kimi Java’nın istifadə faizi olduqca yüksəkdir.
